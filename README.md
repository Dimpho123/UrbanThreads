### E-commerce website (Urban-Threads)
#This is a simple Ecommerce website built using, html, javascript and css. It allows users to browse products, view product details and 
add items to a shopping cart. This project simulates an online store experience.

###Features
---Product listing page
---Shopping cart functionality
---Add or remove items from cart
---Responsive design for mobile and desktop

### It is built with:
---Html
---Javascript
---CSS
---Firebase

###Getting started
---git clone 
---npm install
---npm start

###How it works
--=Users land on homepage showing products
---Clicking a product opens details
---Items can be added to the shopping cart
---Cart updates dynamically
